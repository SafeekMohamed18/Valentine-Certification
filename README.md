# Valentine Certificate

A simple, client-side Valentine’s Day certificate generator built with HTML, CSS and JavaScript.

## Description

This project provides an interactive certificate generator where users can enter a name and custom message, preview a styled certificate, and print or download it. It is fully client-side and requires no backend.

## Features

- Quick customization of recipient name and message
- Responsive, print-friendly layout
- No server required — works in any modern browser

## Files

- `index.html` — main UI and certificate markup
- `style.css` — certificate styling and responsive rules
- `script.js` — form handling, preview, and print/download logic

## Quick start

1. Open `index.html` in your browser to run locally.
2. Or serve the folder with a simple HTTP server:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

## Customize

- Edit `style.css` to change colors, fonts, and layout.
- Edit `script.js` to change text handling, add extra fields or export formats.

## Deploy

Host the repository with GitHub Pages (enable Pages in repository settings) or any static-file host.
